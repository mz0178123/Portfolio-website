## ✨ Features

- **About Me / Navigation Bar**: Clean and fixed header tracking section progress with link names updated to match the profile content exactly (`Profile`, `Work Experience`, `Skills & Tools`).
- **Smooth Navigation Scrolling**: Integrated anchor tags (`#id`) and pure CSS smooth scrolling (`scroll-behavior: smooth`) that automatically slides down to the exact section when clicked.
- **Interactive Dark Mode 🌓**: A beautiful, custom-coded dark/light theme toggle widget using Javascript. Swapping themes updates background gradients, fonts, borders, and accent tags seamlessly without straining the eyes.
- **Responsive Web Design (RWD)**: Optimized for all screens, adapting flawlessly from wide desktop monitors down to mobile viewports using standard CSS Grid and Media Queries.
- **Optimized Avatar Styling**: Rebuilt using the CSS `background-image` layout configuration with `background-size: cover` to completely prevent portrait stretching/distortion while preserving an exact center alignment for the profile image.
- **CEFR B1 English Certification**: Fully updated language proficiency sections.
