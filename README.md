## ✨ 特色功能

- **響應式設計 (Responsive Design)**：無論在電腦還是手機上觀看，排版都能完美適應。
- **漸層背景**：使用現代感十足的柔和漸層配色。
- **互動按鈕**：內建滑鼠懸停動畫與簡單的 JavaScript 彈窗互動。

## 🛠️ 使用技術

- **HTML5**：建構網頁的骨架與內容。
- **CSS3**：負責網頁的視覺外觀、漸層與排版佈局。
- **GitHub Pages**：用於免費託管與發佈靜態網站。
