## 📸 網站風格描述

- **極簡暗黑風格**：採用近年科技圈最流行的深色主題（Dark Mode）。
- **微光漸層效果**：頭貼與按鈕在滑鼠懸停時有動態的霓虹藍光效果（Glow Effect）。
- **毛玻璃質感**：卡片主體使用 `backdrop-filter` 營造出輕透的磨砂玻璃視覺。

## 🛠️ 使用技術

- **HTML5**：網頁結構與內容語意化。
- **CSS3**：負責排版（Flexbox）、漸層配色、毛玻璃特效與動態轉場。
- **GitHub Pages**：網站代管與即時部署。
